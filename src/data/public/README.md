# Public Data

Public JSON files in this folder are safe for the browser to load. They drive the current homepage, biography, collections, museum canon, archive summaries, exhibitions, research, and Support Us sections.

All Chinese fields in public JSON must be Traditional Chinese (`zh-Hant`). The public bilingual system is English + Traditional Chinese for overseas Chinese communities, researchers, museums, universities, and cultural institutions.

Technical IDs, sheet row numbers, and decorative card index numbers should remain internal metadata only. Public rendering may show meaningful years, periods, medium, collection, category, and status when they help museum interpretation.

Use `manifest.json` to see which Google Sheet tab each file came from. Use `archive-index.json` as the future entry point for AI search, archive filtering, collection pages, exhibition pages, and multilingual rendering.
