# Internal Data

Internal JSON files preserve Google Sheet tabs that are useful for planning, review, governance, metadata standards, workflow, funding, reference management, and build priorities.

These files are kept in the repository so no Sheet structure is lost, but the public website does not fetch them yet.

When internal content is promoted to public rendering, Chinese text must follow the project standard: Traditional Chinese (`zh-Hant`) only.

Current internal additions:

- `reference-library.json` preserves the `04_Reference_Library` sheet structure, including reference records, principles, categories, future systems, and long-term reference vision.
- `department-tab-targets.json` preserves department-level ideal tab ranges for future content database planning.
