# Gu Yuan Digital Museum Website Editorial Standards

## Website Editorial Rule #001

No ending `。` or `.` in titles, headings, navigation items, captions, labels, slogans, taglines, gallery names, or footer fragments.

Use punctuation only for complete sentences, paragraph text, quotations, article body text, news content, and formal descriptive text.

This rule applies to both English and Traditional Chinese content. It must be applied to future content additions, translations, page updates, metadata, captions, banners, navigation menus, gallery titles, and footer elements.

Examples:

- Correct: `古元數字美術館`
- Correct: `中國現代記憶的木刻史詩`
- Correct: `The Carved Memory of Modern China`
- Correct: `Explore Collection`
- Incorrect: `古元數字美術館。`
- Incorrect: `中國現代記憶的木刻史詩。`
- Incorrect: `The Carved Memory of Modern China.`
- Incorrect: `Explore Collection.`
